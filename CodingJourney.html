My Coding Journey

It all started when I was playing Growtopia. I was farming this thing called "farmable" which drops gems, and you can use those gems to buy a world lock, which is like the main money in the game. At first, it was fun, but then it got boring. I had to keep planting the seeds after destroying the farmable, waiting for days for them to grow, and then harvesting them. I had like 9000 of these things, and I was like, "Can I even keep doing this?"

Then, I had this idea. I thought, "What if I could find a way to make this farming thing automatic?" So, I searched online for "how to auto farm on Growtopia" and that's when I found this guy called "Chichiya Automation" I tried his software and it worked so well, I was amazed. I got curious about how it all worked, and when I checked the source code, I saw he used Python to do it. I was seriously impressed. I thought, "I want to learn this so I can automate more stuff because, honestly, I'm lazy."

I started learning Python. I tried watching Python tutorials on YouTube, but they were kinda boring, to be honest. Then, I found Codeacademy. I used that for like a month, and I got the basics of Python and some coding stuff. But then, even that got boring.

So, I decided to switch things up. I thought, "Why not learn HTML and CSS?" I watched YouTube videos for that too, did exercises every day, and here I am now. By the way, all of this happened during this summer break. But right now, my plan is to learn JavaScript next. That way, I can add cool things to my website and make it interactive.
